## Édition de base

| Action                | macOS                                      | Windows                        |
| --------------------- | ------------------------------------------ | ------------------------------ |
| **Copier**            | <kbd>⌘</kbd> + <kbd>C</kbd>                | <kbd>Ctrl</kbd> + <kbd>C</kbd> |
| **Coller**            | <kbd>⌘</kbd> + <kbd>V</kbd>                | <kbd>Ctrl</kbd> + <kbd>V</kbd> |
| **Couper**            | <kbd>⌘</kbd> + <kbd>X</kbd>                | <kbd>Ctrl</kbd> + <kbd>X</kbd> |
| **Annuler**           | <kbd>⌘</kbd> + <kbd>Z</kbd>                | <kbd>Ctrl</kbd> + <kbd>Z</kbd> |
| **Refaire**           | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>Z</kbd> | <kbd>Ctrl</kbd> + <kbd>Y</kbd> |
| **Sélectionner tout** | <kbd>⌘</kbd> + <kbd>A</kbd>                | <kbd>Ctrl</kbd> + <kbd>A</kbd> |
| **Rechercher**        | <kbd>⌘</kbd> + <kbd>F</kbd>                | <kbd>Ctrl</kbd> + <kbd>F</kbd> |
| **Remplacer**         | <kbd>⌘</kbd> + <kbd>⌥</kbd> + <kbd>F</kbd> | <kbd>Ctrl</kbd> + <kbd>H</kbd> |
| **Enregistrer**       | <kbd>⌘</kbd> + <kbd>S</kbd>                | <kbd>Ctrl</kbd> + <kbd>S</kbd> |
| **Imprimer**          | <kbd>⌘</kbd> + <kbd>P</kbd>                | <kbd>Ctrl</kbd> + <kbd>P</kbd> |

## Navigation & Applications

| Action               | macOS                                        | Windows                                         |
| -------------------- | -------------------------------------------- | ----------------------------------------------- |
| **Nouvel onglet**    | <kbd>⌘</kbd> + <kbd>T</kbd>                  | <kbd>Ctrl</kbd> + <kbd>T</kbd>                  |
| **Fermer onglet**    | <kbd>⌘</kbd> + <kbd>W</kbd>                  | <kbd>Ctrl</kbd> + <kbd>W</kbd>                  |
| **Rouvrir onglet**   | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>T</kbd>   | <kbd>Ctrl</kbd> + <kbd>⇧</kbd> + <kbd>T</kbd>   |
| **Quitter app**      | <kbd>⌘</kbd> + <kbd>Q</kbd>                  | <kbd>Alt</kbd> + <kbd>F4</kbd>                  |
| **Forcer à quitter** | <kbd>⌥</kbd> + <kbd>⌘</kbd> + <kbd>Esc</kbd> | <kbd>Ctrl</kbd> + <kbd>⇧</kbd> + <kbd>Esc</kbd> |
| **Masquer app**      | <kbd>⌘</kbd> + <kbd>H</kbd>                  | <kbd>Win</kbd> + <kbd>↓</kbd>                   |
| **Minimiser**        | <kbd>⌘</kbd> + <kbd>M</kbd>                  | <kbd>Win</kbd> + <kbd>↓</kbd>                   |

<Callout type="warning">
  **Attention** : Sur macOS, <kbd>⌘</kbd> + <kbd>W</kbd> ferme la fenêtre mais
  pas l'application. Utilisez <kbd>⌘</kbd> + <kbd>Q</kbd> pour quitter
  complètement.
</Callout>
