---
title: Capture d'écran
description: Tous les raccourcis pour capturer l'écran et enregistrer sur macOS
---

| Action                 | Raccourci                                                              |
| ---------------------- | ---------------------------------------------------------------------- |
| **Écran complet**      | <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>3</kbd>                             |
| **Sélection**          | <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>4</kbd>                             |
| **Fenêtre**            | <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>4</kbd> puis <kbd>Espace</kbd>      |
| **Vers presse-papier** | Ajouter <kbd>Ctrl</kbd> au raccourci                                   |
| **Interface capture**  | <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>5</kbd>                             |
| **Enregistrer écran**  | <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>5</kbd> puis choisir enregistrement |
