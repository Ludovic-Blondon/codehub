## Multitâche

| Action                | macOS                                            | Windows                                                      |
| --------------------- | ------------------------------------------------ | ------------------------------------------------------------ |
| **Changer d'app**     | <kbd>⌘</kbd> + <kbd>Tab</kbd>                    | <kbd>Alt</kbd> + <kbd>Tab</kbd>                              |
| **Fenêtres même app** | <kbd>⌘</kbd> + <kbd>`</kbd>                      | <kbd>Alt</kbd> + <kbd>`</kbd>                                |
| **Mission Control**   | <kbd>F3</kbd> ou <kbd>Ctrl</kbd> + <kbd>↑</kbd>  | <kbd>Win</kbd> + <kbd>Tab</kbd>                              |
| **Afficher bureau**   | <kbd>F11</kbd> ou <kbd>Fn</kbd> + <kbd>F11</kbd> | <kbd>Win</kbd> + <kbd>D</kbd>                                |
| **Plein écran**       | <kbd>Ctrl</kbd> + <kbd>⌘</kbd> + <kbd>F</kbd>    | <kbd>F11</kbd>                                               |
| **Changer d'espace**  | <kbd>Ctrl</kbd> + <kbd>←</kbd>/<kbd>→</kbd>      | <kbd>Ctrl</kbd> + <kbd>Win</kbd> + <kbd>←</kbd>/<kbd>→</kbd> |

## Gestion des fenêtres (macOS Sequoia 15+)

| Action               | Raccourci                                      |
| -------------------- | ---------------------------------------------- |
| **Carrelage gauche** | <kbd>Fn</kbd> + <kbd>Ctrl</kbd> + <kbd>←</kbd> |
| **Carrelage droite** | <kbd>Fn</kbd> + <kbd>Ctrl</kbd> + <kbd>→</kbd> |
| **Remplir écran**    | <kbd>Fn</kbd> + <kbd>Ctrl</kbd> + <kbd>F</kbd> |
| **Centrer fenêtre**  | <kbd>Fn</kbd> + <kbd>Ctrl</kbd> + <kbd>C</kbd> |
