## Navigation et répertoires

| ------- | ------------------ | ---------------------------------- |
| `~`     | `cd ~`             | Aller au répertoire home           |

### Exemples d'utilisation

```bash
# Navigation rapide
... # équivaut à cd ../..
d   # voir les répertoires récents
2   # aller au 2ème répertoire de la pile

# Git workflow typique
gst      # git status
ga .     # git add .
gcmsg "mon commit"  # git commit --message "mon commit"
gp       # git push

# Branches Git
gcb feature/nouvelle-fonctionnalite  # créer et changer de branche
gco main    # retourner sur main
gm feature/nouvelle-fonctionnalite   # merger la branche
```

> 🔴 **Remarque**
>
> Ces alias sont automatiquement disponibles avec Oh My Zsh. Pour voir tous les alias disponibles, tapez `alias` dans votre terminal.
> Les alias Git nécessitent d'avoir le plugin git activé dans votre configuration Oh My Zsh.
