---
title: Édition de texte avancée
description: Raccourcis pour la navigation et l'édition de texte avancée sur macOS
---

## Navigation

| Action             | macOS                        | Windows                           |
| ------------------ | ---------------------------- | --------------------------------- |
| **Début de ligne** | <kbd>⌘</kbd> + <kbd>←</kbd>  | <kbd>Home</kbd>                   |
| **Fin de ligne**   | <kbd>⌘</kbd> + <kbd>→</kbd>  | <kbd>End</kbd>                    |
| **Début document** | <kbd>⌘</kbd> + <kbd>↑</kbd>  | <kbd>Ctrl</kbd> + <kbd>Home</kbd> |
| **Fin document**   | <kbd>⌘</kbd> + <kbd>↓</kbd>  | <kbd>Ctrl</kbd> + <kbd>End</kbd>  |
| **Mot précédent**  | <kbd>⌥</kbd> + <kbd>←</kbd>  | <kbd>Ctrl</kbd> + <kbd>←</kbd>    |
| **Mot suivant**    | <kbd>⌥</kbd> + <kbd>→</kbd>  | <kbd>Ctrl</kbd> + <kbd>→</kbd>    |
| **Page haut**      | <kbd>Fn</kbd> + <kbd>↑</kbd> | <kbd>Page Up</kbd>                |
| **Page bas**       | <kbd>Fn</kbd> + <kbd>↓</kbd> | <kbd>Page Down</kbd>              |

## Sélection

| Action                               | macOS                                                   | Windows                                                    |
| ------------------------------------ | ------------------------------------------------------- | ---------------------------------------------------------- |
| **Sélectionner mot**                 | <kbd>⌥</kbd> + <kbd>⇧</kbd> + <kbd>←</kbd>/<kbd>→</kbd> | <kbd>Ctrl</kbd> + <kbd>⇧</kbd> + <kbd>←</kbd>/<kbd>→</kbd> |
| **Sélectionner ligne**               | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>←</kbd>/<kbd>→</kbd> | <kbd>⇧</kbd> + <kbd>Home</kbd>/<kbd>End</kbd>              |
| **Sélectionner tout jusqu'au début** | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>↑</kbd>              | <kbd>Ctrl</kbd> + <kbd>⇧</kbd> + <kbd>Home</kbd>           |
| **Sélectionner tout jusqu'à la fin** | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>↓</kbd>              | <kbd>Ctrl</kbd> + <kbd>⇧</kbd> + <kbd>End</kbd>            |

## Suppression

| Action                       | macOS                                            | Windows                                               |
| ---------------------------- | ------------------------------------------------ | ----------------------------------------------------- |
| **Supprimer mot gauche**     | <kbd>⌥</kbd> + <kbd>Delete</kbd>                 | <kbd>Ctrl</kbd> + <kbd>Backspace</kbd>                |
| **Supprimer mot droite**     | <kbd>Fn</kbd> + <kbd>⌥</kbd> + <kbd>Delete</kbd> | <kbd>Ctrl</kbd> + <kbd>Delete</kbd>                   |
| **Supprimer jusqu'au début** | <kbd>⌘</kbd> + <kbd>Delete</kbd>                 | <kbd>Ctrl</kbd> + <kbd>⇧</kbd> + <kbd>Backspace</kbd> |
