# ⌨️ Raccourcis clavier macOS

Découvrez les raccourcis clavier les plus utiles sur macOS, du plus **pertinent** au plus **avancé**.  
Ces astuces vous aideront à **gagner du temps** et à améliorer votre productivité au quotidien.

---

## 🟢 Les incontournables (à connaître absolument)

- **Copier** : `⌘ C`
- **Coller** : `⌘ V`
- **Couper** : `⌘ X`
- **Annuler** : `⌘ Z`
- **Revenir en arrière (dans Safari/Finder)** : `⌘ [`
- **Aller de l’avant** : `⌘ ]`
- **Rechercher** : `⌘ F`
- **Nouvel onglet (navigateur/Finder)** : `⌘ T`
- **Fermer la fenêtre** : `⌘ W`
- **Fermer l'application** : `⌘ Q`
- **Forcer à quitter** : `⌥ ⌘ ⎋`

---

## 🟡 Navigation & gestion des fenêtres

- **Changer d’application** : `⌘ ⇥`
- **Changer de fenêtre dans la même app** : `⌘ ˜` (tilde)
- **Mission Control** : `F3` ou `⌃ ↑`
- **Bureau** : `F11` ou `⌘ F3`
- **Plein écran** : `⌃ ⌘ F`
- **Diviser l’écran (Split View, souris)** : maintenir le bouton vert de fenêtre

### 🚀 Nouveaux raccourcis d’organisation des fenêtres (macOS Ventura+)

| Icône | Nom          | Raccourci clavier | Description                                                     |
| ----- | ------------ | ----------------- | --------------------------------------------------------------- |
| 🖥️    | **Remplir**  | `Fn + ⌃ F`        | Agrandir la fenêtre active pour remplir le bureau.              |
| 🎯    | **Centrer**  | `Fn + ⌃ C`        | Centrer la fenêtre active sur le bureau.                        |
| ⬅️    | **À gauche** | `Fn + ⌃ ←`        | Déplacer la fenêtre active vers la partie gauche du bureau.     |
| ➡️    | **À droite** | `Fn + ⌃ →`        | Déplacer la fenêtre active vers la partie droite du bureau.     |
| ⬆️    | **En haut**  | `Fn + ⌃ ↑`        | Déplacer la fenêtre active vers la partie supérieure du bureau. |
| ⬇️    | **En bas**   | `Fn + ⌃ ↓`        | Déplacer la fenêtre active vers la partie inférieure du bureau. |

---

## 🔵 Productivité & édition de texte

- **Sélectionner tout** : `⌘ A`
- **Sélectionner un mot** : `⌥ ⇧ →` / `⌥ ⇧ ←`
- **Aller au début de la ligne** : `⌘ ←`
- **Aller à la fin de la ligne** : `⌘ →`
- **Aller au début du document** : `⌘ ↑`
- **Aller à la fin du document** : `⌘ ↓`
- **Supprimer le mot précédent** : `⌥ ⌫`
- **Supprimer le mot suivant** : `Fn ⌥ ⌫`

---

## 🟣 Capture d’écran & outils intégrés

- **Capture de tout l’écran** : `⇧ ⌘ 3`
- **Capture d’une sélection** : `⇧ ⌘ 4`
- **Capture d’une fenêtre** : `⇧ ⌘ 4`, puis `Espace`
- **Capture et copie dans presse-papier** : ajouter `Ctrl` (`⇧ ⌘ Ctrl 4`)
- **Spotlight** : `⌘ Space`
- **Launchpad** : `F4` ou `⌘ F4`
- **Emojis & symboles** : `⌃ ⌘ Space`

---

## 🔴 Avancés & power users

- **Afficher le Dock immédiatement** : `⌥ ⌘ D`
- **Masquer toutes les fenêtres sauf l’actuelle** : `⌥ ⌘ H`
- **Changer de bureau (Spaces)** : `⌃ ←` ou `⌃ →`
- **Rechercher dans Finder** : `⌘ Space`, puis taper le nom du fichier
- **Aperçu rapide d’un fichier** : sélectionner et appuyer sur `Espace`
- **Afficher chemin dans Finder** : `⌘ ⌥ P`
- **Nouveau dossier avec sélection** : `⌘ ⌥ N`

---

## 🧑‍💻 Tips bonus

- Maintenir **`⌥`** (Option/Alt) : débloque des options cachées dans les menus.
- Maintenir **`⇧`** (Shift) : souvent inverse le sens d’un raccourci (ex : défilement lent).
- Maintenir **`⌘ Tab`** et rester appuyé sur `⌘` : naviguer dans les apps avec `Tab` (ou `← / →`).

---

👉 Ces raccourcis sont organisés pour accompagner ta progression. Tu peux commencer par les basiques et intégrer progressivement les plus avancés.
