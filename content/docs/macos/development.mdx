## Navigation

| Action               | macOS                                      | Windows                                       |
| -------------------- | ------------------------------------------ | --------------------------------------------- |
| **Quick Open**       | <kbd>⌘</kbd> + <kbd>P</kbd>                | <kbd>Ctrl</kbd> + <kbd>P</kbd>                |
| **Command Palette**  | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>P</kbd> | <kbd>Ctrl</kbd> + <kbd>⇧</kbd> + <kbd>P</kbd> |
| **Go to Symbol**     | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>O</kbd> | <kbd>Ctrl</kbd> + <kbd>⇧</kbd> + <kbd>O</kbd> |
| **Go to Line**       | <kbd>Ctrl</kbd> + <kbd>G</kbd>             | <kbd>Ctrl</kbd> + <kbd>G</kbd>                |
| **Go to Definition** | <kbd>F12</kbd>                             | <kbd>F12</kbd>                                |
| **Peek Definition**  | <kbd>⌥</kbd> + <kbd>F12</kbd>              | <kbd>Alt</kbd> + <kbd>F12</kbd>               |
| **Find References**  | <kbd>⇧</kbd> + <kbd>F12</kbd>              | <kbd>⇧</kbd> + <kbd>F12</kbd>                 |
