Découvrez les raccourcis clavier les plus utiles sur macOS, du plus **pertinent** au plus **avancé**.  
Ces astuces vous aideront à **gagner du temps** et à améliorer votre productivité au quotidien.

---

## 🟢 Les incontournables (à connaître absolument)

- **Copier** : <kbd>⌘</kbd> + <kbd>C</kbd>
- **Coller** : <kbd>⌘</kbd> + <kbd>V</kbd>
- **Couper** : <kbd>⌘</kbd> + <kbd>X</kbd>
- **Annuler** : <kbd>⌘</kbd> + <kbd>Z</kbd>
- **Revenir en arrière (dans Safari/Finder)** : <kbd>⌘</kbd> + <kbd>[</kbd>
- **Aller de l'avant** : <kbd>⌘</kbd> + <kbd>]</kbd>
- **Rechercher** : <kbd>⌘</kbd> + <kbd>F</kbd>
- **Nouvel onglet (navigateur/Finder)** : <kbd>⌘</kbd> + <kbd>T</kbd>
- **Fermer la fenêtre** : <kbd>⌘</kbd> + <kbd>W</kbd>
- **Fermer l'application** : <kbd>⌘</kbd> + <kbd>Q</kbd>
- **Forcer à quitter** : <kbd>⌥</kbd> + <kbd>⌘</kbd> + <kbd>⎋</kbd>

---

## 🟡 Navigation & gestion des fenêtres

- **Changer d'application** : <kbd>⌘</kbd> + <kbd>⇥</kbd>
- **Changer de fenêtre dans la même app** : <kbd>⌘</kbd> + <kbd>˜</kbd> (tilde)
- **Mission Control** : <kbd>F3</kbd> ou <kbd>⌃</kbd> + <kbd>↑</kbd>
- **Bureau** : <kbd>F11</kbd> ou <kbd>⌘</kbd> + <kbd>F3</kbd>
- **Plein écran** : <kbd>⌃</kbd> + <kbd>⌘</kbd> + <kbd>F</kbd>
- **Diviser l'écran (Split View, souris)** : maintenir le bouton vert de fenêtre

| Icône | Nom          | Raccourci clavier                           | Description                                                     |

| 🖥️    | **Remplir**  | <kbd>Fn</kbd> + <kbd>⌃</kbd> + <kbd>F</kbd> | Agrandir la fenêtre active pour remplir le bureau.              |
| 🎯    | **Centrer**  | <kbd>Fn</kbd> + <kbd>⌃</kbd> + <kbd>C</kbd> | Centrer la fenêtre active sur le bureau.                        |

| ➡️    | **À droite** | <kbd>Fn</kbd> + <kbd>⌃</kbd> + <kbd>→</kbd> | Déplacer la fenêtre active vers la partie droite du bureau.     |
| ⬆️    | **En haut**  | <kbd>Fn</kbd> + <kbd>⌃</kbd> + <kbd>↑</kbd> | Déplacer la fenêtre active vers la partie supérieure du bureau. |

## 🔵 Productivité & édition de texte

- **Sélectionner tout** : <kbd>⌘</kbd> + <kbd>A</kbd>
- **Sélectionner un mot** : <kbd>⌥</kbd> + <kbd>⇧</kbd> + <kbd>→</kbd> / <kbd>⌥</kbd> + <kbd>⇧</kbd> + <kbd>←</kbd>
- **Aller au début de la ligne** : <kbd>⌘</kbd> + <kbd>←</kbd>
- **Aller à la fin de la ligne** : <kbd>⌘</kbd> + <kbd>→</kbd>
- **Aller au début du document** : <kbd>⌘</kbd> + <kbd>↑</kbd>
- **Aller à la fin du document** : <kbd>⌘</kbd> + <kbd>↓</kbd>
- **Supprimer le mot précédent** : <kbd>⌥</kbd> + <kbd>⌫</kbd>
- **Supprimer le mot suivant** : <kbd>Fn</kbd> + <kbd>⌥</kbd> + <kbd>⌫</kbd>

---

## 🟣 Capture d'écran & outils intégrés

- **Capture de tout l'écran** : <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>3</kbd>
- **Capture d'une sélection** : <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>4</kbd>
- **Capture d'une fenêtre** : <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>4</kbd>, puis <kbd>Espace</kbd>
- **Capture et copie dans presse-papier** : ajouter <kbd>⌃</kbd> (<kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>⌃</kbd> + <kbd>4</kbd>)
- **Spotlight** : <kbd>⌘</kbd> + <kbd>Espace</kbd>
- **Launchpad** : <kbd>F4</kbd> ou <kbd>⌘</kbd> + <kbd>F4</kbd>
- **Emojis & symboles** : <kbd>⌃</kbd> + <kbd>⌘</kbd> + <kbd>Espace</kbd>

---

## 🔴 Avancés & power users

- **Afficher le Dock immédiatement** : <kbd>⌥</kbd> + <kbd>⌘</kbd> + <kbd>D</kbd>
- **Masquer toutes les fenêtres sauf l'actuelle** : <kbd>⌥</kbd> + <kbd>⌘</kbd> + <kbd>H</kbd>
- **Changer de bureau (Spaces)** : <kbd>⌃</kbd> + <kbd>←</kbd> ou <kbd>⌃</kbd> + <kbd>→</kbd>
- **Rechercher dans Finder** : <kbd>⌘</kbd> + <kbd>Espace</kbd>, puis taper le nom du fichier
- **Aperçu rapide d'un fichier** : sélectionner et appuyer sur <kbd>Espace</kbd>
- **Afficher chemin dans Finder** : <kbd>⌘</kbd> + <kbd>⌥</kbd> + <kbd>P</kbd>
- **Nouveau dossier avec sélection** : <kbd>⌘</kbd> + <kbd>⌥</kbd> + <kbd>N</kbd>

---

## 🧑‍💻 Tips bonus

- Maintenir **<kbd>⌥</kbd>** (Option/Alt) : débloque des options cachées dans les menus.
- Maintenir **<kbd>⇧</kbd>** (Shift) : souvent inverse le sens d'un raccourci (ex : défilement lent).
- Maintenir **<kbd>⌘</kbd> + <kbd>Tab</kbd>** et rester appuyé sur <kbd>⌘</kbd> : naviguer dans les apps avec <kbd>Tab</kbd> (ou <kbd>←</kbd> / <kbd>→</kbd>).
